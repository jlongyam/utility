## Info

### Class name for viewport

`
$media-xx-small:[state]:[...]:[property]:[value]
$media-x-small:[IDEM]
$media-small:[IDEM]
$media-medium:[IDEM]
$media-large:[IDEM]
$media-x-large:[IDEM]
$media-xx-large:[IDEM]
`

### More

- use _ as SPACE
- state max 5: $name:state1:state2:state3:proerty:value

## Versioning

`[Major].[Minor].[Patch]`

- __Patch__ increments every time you release a build.
- __Minor__ increments every time backwards compatible functionality is added.
- __Major__ increments when new functionality is not backwards compatible.
